# Hospital Emergency Room Dashboard

A comprehensive monthly performance dashboard for hospital emergency department analytics, built in Excel with advanced visualizations and KPI tracking.

## 📊 Project Overview

This dashboard provides real-time insights into hospital emergency room operations through interactive visualizations and key performance indicators. It enables hospital administrators and staff to monitor patient flow, satisfaction rates, and operational efficiency.

### Key Features

- **Patient Volume Tracking**: Monitor total patient admissions and trends across months
- **Wait Time Analytics**: Average wait times visualization to identify bottlenecks
- **Patient Satisfaction Metrics**: Real-time satisfaction scores (0-5 scale)
- **Admission Status Breakdown**: Clear view of admitted vs. non-admitted patients
- **Demographics Analysis**: 
  - Gender distribution (Male/Female)
  - Age group patient distribution (0-9 through 70-79)
- **Department Referrals**: Track patient distribution across 7+ departments
- **Temporal Analysis**: Month-by-month performance tracking
